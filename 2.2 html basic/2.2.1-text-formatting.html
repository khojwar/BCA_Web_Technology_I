Text Formatting Elements in HTML

Bold text: This is an example of bold text.

Important text: It is important to submit the report on time.

Italic text: This is an example of italicized text.

Emphasized text: I really need to focus on this task.

Marked text: Please highlight the important points.

Small text: This is an example of small text.

Deleted text: This item was removed from the list.

Inserted text: This item was added to the list.

Subscript text: H2O is the chemical formula for water.

Superscript text: Einstein's equation is E = mc2.
